# Accountability Wellness App

## Overview

The Accountability Wellness App is a comprehensive tool designed to help users track their wellness journey. It provides a platform for users to set goals, monitor progress, and stay motivated in their personal health and wellness objectives.

## Features
- **Goal Setting**: Users can set short-term and long-term wellness goals.
- **Progress Tracking**: Visualize progress through charts and statistics.
- **Motivational Reminders**: Get reminders to stay on track with your goals.
- **Community Support**: Connect with others for encouragement and accountability.
- **Resources**: Access articles and tips for health and wellness.

## Technologies Used
- **Frontend**: React.js
- **Backend**: Node.js with Express
- **Database**: MongoDB

## Installation
1. Clone the repository: `git clone https://github.com/sjacobs-creator/accountability-wellness-app.git`
2. Navigate to the project directory: `cd accountability-wellness-app`
3. Install dependencies: `npm install`
4. Start the development server: `npm start`
